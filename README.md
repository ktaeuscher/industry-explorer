# Industry Attractiveness Explorer

An interactive tool for exploring how industries differ in profitability, growth, and return on investment. Built for educational use in strategy and industry analysis courses.

**[Launch the tool](https://karltaeuscher.github.io/industry-explorer/)**

## About

This tool allows students to compare 94 global industries across three dimensions:

- **Profit Margin** — Operating margin (share of revenue retained as profit)
- **Industry Growth** — Revenue CAGR over 5 years
- **Return on Investment** — Return on invested capital (ROIC)

Data source: [Aswath Damodaran, NYU Stern](https://pages.stern.nyu.edu/~adamodar/New_Home_Page/datacurrent.html). Global publicly traded firms, 2011–2026.

## Features

- Bar chart, scatter plot, and sortable data table views
- Click any industry for a detailed profile with historical trends
- Percentile rankings and comparison to market averages
- Fully self-contained — no server or dependencies required
